# Идеи дальнейшего развития

Что ещё можно сделать из P1-Nano пультом для мака. Сгруппировано по сложности.
Помечено: ✅ легко (часы), 🟡 средне, 🔴 сложно/нужны приватные API.

## Звук и медиа
- ✅ **Мьют микрофона** (кнопка) — toggle на default input device (CoreAudio
  `kAudioDevicePropertyMute`, scope Input). Очень удобно для созвонов (Zoom/Meet).
- ✅ **Громкость микрофона / gain** на энкодере (input VolumeScalar).
- 🟡 **Громкость отдельных приложений** на энкодерах (YouTube отдельно от Spotify) —
  через новый CoreAudio process API (`kAudioHardwarePropertyProcessObjectList`, macOS 14.4+).
- 🟡 **Перемотка видео** энкодером — YouTube/плеер: ← / → (стрелки) по тикам, или J/L.
- 🟡 **Now Playing на LCD** — показывать на экране устройства название текущего трека
  (Music/Spotify через `MediaRemote` или AppleScript).

## VU-метры (то, что ты спрашивал)
- 🔴 **Реальный уровень звука на индикаторах устройства.** P1-Nano умеет рисовать метры
  (`D0` Channel Pressure / `14 20`). Чтобы кормить их настоящим уровнем воспроизведения,
  нужно перехватить системный звук: **CoreAudio Process Tap** (`AudioHardwareCreateProcessTap`,
  macOS 14.4+) — снимаем выходной поток, считаем RMS, шлём на метры/мотор. Без сторонних
  драйверов. Это самостоятельный мини-проект, но реальный.

## Система и навигация
- ✅ **Переключение рабочих столов** (кнопки) — Ctrl+← / Ctrl+→ (CGEvent с клавишами).
- ✅ **Mission Control / Launchpad / Показать рабочий стол** (кнопки) — спец-клавиши.
- ✅ **Скриншот / запись экрана** (кнопки) — Cmd+Shift+4 / Cmd+Shift+5.
- ✅ **Focus / Не беспокоить** toggle (кнопка) — через Shortcuts CLI (`shortcuts run …`).
- ✅ **Яркость клавиатуры** на энкодере (медиа-клавиши подсветки).
- 🟡 **Яркость внешнего монитора** — медиа-клавиши берут только встроенный; внешний — по
  DDC/CI (утилита `m1ddc`/BetterDisplay).
- 🟡 **Управление окнами** (кнопки) — раскладка окон через yabai/Rectangle CLI.

## Макросы и приложения
- ✅ **Любая кнопка → AppleScript / shell / URL** — расширить `enum Action` кейсом
  `.run("команда")`: открыть сайт, запустить Shortcut, вставить текст, что угодно.
- ✅ **Тачскрин (80 кнопок) → запуск приложений / макросы.** В iMap назначить кнопкам
  тачскрина MIDI-ноты (или hotkey-HID), затем мэппить их в хосте как обычные кнопки.
  Требует один раз отредактировать конфиг устройства (бэкап уже есть).
- 🟡 **Профили под приложение** — авто-переназначение кнопок в зависимости от активного
  приложения (front app через NSWorkspace): в Ableton одно, в браузере другое.

## Удобство
- ✅ **Точная подстройка кривой фейдера** (лог/линейная) и зон 0 %/100 %.
- ✅ **Сохранение/восстановление громкости** по кнопке (запомнить уровень, вернуть).
- 🟡 **Меню в строке статуса** (NSStatusItem) — иконка, статус подключения, вкл/выкл,
  переключение профилей, без терминала.
