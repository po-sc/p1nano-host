# Идеи дальнейшего развития

Что ещё можно сделать из P1-Nano пультом для мака. Помечено: ✅ легко (часы),
🟡 средне, 🔴 сложно/нужны приватные API. Многое уже мапится одной строкой в
`mapping.conf` через `run <команда>` — для таких отдельный код не нужен.

## Готово (для контекста)
Фейдер↔громкость (двусторонне) · HUD громкости/яркости · LCD-индикатор на iCON ·
Play/Pause/треки (MediaRemote) · Mute звука (кнопка 16) · Mute микрофона (кнопка 8/Solo) ·
яркость и громкость на энкодерах · запуск приложений · макросы `run` · автозапуск ·
авто-переподключение · конфиг без пересборки · настройка сенсорного экрана по нотам.

## ТОП — что добавить следующим (приоритет)
1. 🟡 **Now Playing на экране iCON** — показывать название текущего трека/исполнителя на
   LCD устройства (через `MRMediaRemoteGetNowPlayingInfo`). Самое «вау» из дешёвого.
2. 🟡 **Громкость отдельных приложений** на энкодерах — YouTube/Spotify/Zoom по отдельности
   (CoreAudio process API `kAudioHardwarePropertyProcessObjectList`, macOS 14.4+).
3. 🟡 **Статус-меню в строке меню** (NSStatusItem) — иконка с состоянием (подключён/нет),
   вкл/выкл, переключение профилей — управление без терминала.
4. 🟡 **Профили под активное приложение** — авто-смена привязок по фронт-приложению
   (NSWorkspace): в Ableton одно, в браузере другое.

## Через `run` — уже доступно, просто добавь строку в mapping.conf
- ✅ **Caffeinate** (не спать): `run caffeinate -t 3600`
- ✅ **Погасить экран / лок**: `run pmset displaysleepnow`
- ✅ **Скриншот**: `run screencapture -i ~/Desktop/shot.png`
- ✅ **Focus / Не беспокоить**: `run shortcuts run "Focus"`
- ✅ **Открыть сайт/папку**: `run open https://...` / `run open ~/Downloads`
- ✅ **Переключение столов**: `run osascript -e '...key code 124 using control down'`
  (через AppleScript System Events; нужен Accessibility у `osascript`)
- ✅ **Mission Control / Launchpad**: `run open -a "Mission Control"`

## Звук и медиа
- ✅ **Громкость/gain микрофона** на энкодере (input `VolumeScalar`).
- 🟡 **Перемотка видео** энкодером — ← / → или J/L по тикам.
- 🔴 **VU-метры** — реальный уровень звука на индикаторах/моторе устройства, через
  CoreAudio Process Tap (`AudioHardwareCreateProcessTap`, macOS 14.4+): снять выход,
  посчитать RMS, слать на метры. Самостоятельный мини-проект.

## Система / экраны
- 🟡 **Яркость внешнего монитора** — медиа берут только встроенный; внешний по DDC (`m1ddc`).
- ✅ **Яркость клавиатуры** на энкодере (DisplayServices keyboard brightness / медиа-клавиша).
- 🟡 **Управление окнами** (раскладка) — через yabai/Rectangle CLI в `run`.

## Удобство
- ✅ **Сохранить/вернуть громкость** по кнопке (запомнить уровень).
- ✅ **Тонкая кривая фейдера** (лог/линейная), зоны 0/100 %.
- 🟡 **Горячая перезагрузка конфига** — следить за `mapping.conf` и применять без рестарта.
